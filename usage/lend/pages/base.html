#paragraph

The terms Internet and World Wide Web are often used without much distinction. However, the Internet and the World Wide Web are not the same. The Internet is a global system of interconnected computer networks using standard protocols. In contrast, the "Web" is one of the services transferred over these networks using these protocols. It is a collection of text documents and other resources, linked by hyperlinks and URLs, usually accessed by web browsers from web servers

The primary function of a web server is to deliver web pages in response to client requests. This means delivery of HTML documents and any additional content that may be included by a document, such as images, style sheets and scripts.
